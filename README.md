# Field · Theory

An interactive textbook on classical electromagnetism, structured as **sixteen
labs across four chapters**. One equation per lab. Each lab has sliders to
play with the variables, a real-time canvas visualization, live-computed
outputs, a long-form deep dive that explains *why* the formula has the form
it does, and a **per-page Sources section** so every numerical claim and every
historical attribution traces back to a primary source.

```
Chapter I  · Electric Field    1.1  F = kQ₁Q₂/r²        Coulomb's law
                               1.2  E = kQ/r²            Field of a point charge
                               1.3  ∮E·dA = Q_enc/ε₀    Gauss's law
                               1.4  V = −∫E·dℓ          Potential difference

Chapter II · Magnetic Field    2.1  dB = (μ₀/4π)Idℓ×r̂/r²   Biot–Savart
                               2.2  ∮B·dℓ = μ₀ I_enc    Ampère's law
                               2.3  F = q(E + v×B)      Lorentz force
                               2.4  EMF = −dΦ_B/dt      Faraday's law

Chapter III · Conduction       3.1  J = σE              Ohm's law (microscopic)
                               3.2  R = ρL/A            Resistance
                               3.3  v_d = I/(nqA)       Drift velocity
                               3.4  P = I²R             Joule heating

Chapter IV · Energy & Fields   4.1  C = ε₀εᵣA/d         Capacitance
                               4.2  V = −L dI/dt        Inductance
                               4.3  u = ½ε₀E² + B²/2μ₀  Field energy density
                               4.4  S = E×B/μ₀          Poynting vector
```

## Stack

- **React 18** + **TypeScript** (strict) + **Vite 5** + **TanStack Router**
- Vanilla canvas viz (no charting/graphing libs — every visualization is
  hand-drawn so it can match the typography of the rest of the site)
- No CSS-in-JS, no Tailwind — a single `src/styles/main.css` with design tokens

## Sourcing rule

Every numerical claim, every historical attribution, every quoted line is
cited to a primary or canonical secondary source listed in
`src/lib/sources.ts`. Inline citations look like `[3]` pills and link down
to the per-page Sources block. **No claim ships without a source key. No
source key without a real, verified reference.**

Constants come from CODATA 2018 (NIST). Material conductivities from the CRC
Handbook 104th ed. Free-electron densities from Ashcroft & Mermin. Standard
derivations cite Griffiths (4th ed.) or Jackson (3rd ed.). Historical claims
cite the original Coulomb (1785), Cavendish (1773), Biot–Savart (1820),
Ampère (1826), Faraday (1832), Maxwell (1865), Poynting (1884), Drude (1900),
Joule (1841), Hall (1879), Williams–Faller–Hill (1971) papers as appropriate.

## How to add a new lab

1. Add a manifest entry to `src/labs/data/manifest.ts` (slug, chapter, formula, blurb, hero).
2. Add a sources key list to `BASE_LAB_SOURCES` in the same file. If you need a new source, add it to `src/lib/sources.ts` with a real URL/venue.
3. Create `src/labs/{YourLab}.tsx`. Mirror `CoulombLab.tsx`:
   - useState for inputs, useMemo for computed outputs, useCallback for canvas setup.
   - Wrap with `<LabShell slug="…" />` and pass `labContent` (the canvas + sliders + readouts) and `prose`.
   - Inline citations: `<Cite id="some-source-key" in={SOURCES} />`.
4. Register the slug in the lazy-import map in `src/routes/labs.$slug.tsx`.
